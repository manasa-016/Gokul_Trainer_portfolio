# 🚀 Gokulakrishnan Muthusamy — Portfolio

A stunning, dark-themed 3D portfolio showcasing expertise in AI/ML, Data Science, and Training.

## ✨ Features

- **3D Particle Background** — Interactive Three.js particles with connection lines
- **Dark Premium Theme** — Glassmorphism, gradients, and glowing accents
- **3D Card Tilt Effects** — Mouse-tracking perspective transforms
- **Custom Cursor** — Glowing dot cursor with hover effects
- **Typing Animation** — Dynamic role display
- **Scroll Animations** — Intersection Observer-based reveals
- **Smooth Counter** — Animated statistics
- **Fully Responsive** — Mobile-first design
- **Contact Form** — Mailto integration

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript ES6+
- Three.js (3D particle background)
- Google Fonts (Outfit, Inter, JetBrains Mono)
- Font Awesome Icons

## 🎯 Sections

1. **Hero** — Introduction, stats, and 3D profile card
2. **About** — Background with code-style visual
3. **Experience** — Timeline of professional journey
4. **Skills** — Technical arsenal with proficiency levels
5. **Education** — Academic background and certifications
6. **Contact** — Form and social links

## 🚀 Getting Started

Simply open `index.html` in a browser, or serve it locally:

```bash
python -m http.server 3000
# Open http://localhost:3000
```
